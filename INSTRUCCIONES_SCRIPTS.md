# 📜 Instrucciones de Uso de los Scripts

He creado dos scripts para facilitar la ejecución del programa sin tener que usar comandos complejos:

### 1. `RUN_DASHBOARD.bat`
**Uso**: Haz doble clic sobre este archivo.
- **Qué hace**: Activa automáticamente el entorno virtual y lanza el Dashboard en tu navegador.
- **Ideal para**: Ver resultados, procesar auditorías de forma visual y descargar informes Excel.

### 2. `RUN_ENGINE_CLI.bat`
**Uso**: Haz doble clic sobre este archivo.
- **Qué hace**: Te pedirá el nombre del **Cliente** y la **Fecha**. Luego ejecutará el motor de cálculo directamente en la consola.
- **Ideal para**: Procesamientos rápidos si ya conoces los datos y no necesitas la interfaz visual.

---
> [!NOTE]
> Para que funcionen, asegúrate de que la carpeta `.venv` exista (puedes crearla siguiendo la `GUIA_INSTALACION.md`).
